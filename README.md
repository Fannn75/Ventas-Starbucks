# Análisis de Ventas – Starbucks (Ene–Abr 2025)

Proyecto de análisis de datos desarrollado en Power BI con el objetivo de evaluar el desempeño comercial y el comportamiento de clientes durante el período enero–abril 2025.

##  Objetivo

Analizar indicadores clave de ventas, identificar productos estratégicos, evaluar la evolución mensual y explorar oportunidades de mejora orientadas a la toma de decisiones basada en datos.

##  Alcance del análisis

El análisis contempla la evaluación del desempeño comercial mediante indicadores clave de ventas, evolución mensual, comportamiento por producto y sucursal, así como la segmentación de clientes con y sin membresía. Asimismo, se identifican patrones de consumo y oportunidades estratégicas para optimizar la rentabilidad.

##  Herramientas utilizadas

- Power BI  
- Modelado de datos  
- Análisis exploratorio  

##  Archivos incluidos

- Archivo Power BI (.pbix)  
- Reporte completo en PDF  
- Dashboard  

## ⚠️ Nota

Los datos utilizados en este reporte son completamente ficticios y fueron creados con fines educativos y demostrativos. No representan información real de Starbucks ni de ninguna otra empresa.
